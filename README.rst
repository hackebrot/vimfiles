Cheat Sheet

Count number of search matches
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

The following will display the count, but will not change the buffer::

    :%s/pattern//gn

See `Tip 860`_ by Marc Weber on the `vim wiki`_.
